# Project 2: Network Reconnaissance and Vulnerability Assessment Lab

## Project Overview
This project demonstrates a controlled network reconnaissance and vulnerability assessment lab using Nmap and Wireshark. The goal is to identify a lab target, discover open ports and services, observe basic network traffic, map findings to potential risks, and document remediation recommendations.

## Lab Environment
- Analyst machine: MacBook Pro
- Target machine: Ubuntu-SOC-Lab VM in UTM
- Target IP address: 192.168.64.2
- Tools: Nmap, Wireshark, Terminal
- Scope: Local lab environment only

## Project Objectives
- Connect to the Ubuntu lab VM using SSH
- Identify the target VM IP address
- Perform host discovery
- Scan for open ports
- Detect service versions
- Capture and review basic network traffic
- Document risks, business impact, and remediation steps

## Repository Structure
- screenshots/: Evidence screenshots from each step
- scans/: Nmap scan outputs
- pcaps/: Wireshark packet captures
- notes/: Working notes
- report/: Final vulnerability assessment report
